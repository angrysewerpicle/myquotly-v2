# MyQuotly
### a simple buisness that makes getting online quotes easy

what to add to .env:

<br>
<div>
    DATABASE_URL=(link to mongodb)
</div>
<div>
    AUTH_EMAIL=(email u want to use)
</div>
<div>
    AUTH_PASS=(email pass - uses nodemailer)
</div>
<div>
    EMAIL_SEND_URL=(current root url to website)
</div>
